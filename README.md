# Portfolio — Athéna D.

Portfolio en ligne présentant mon parcours en administration systèmes, réseaux et cybersécurité (étudiante B3 Cybersécurité, en recherche d'alternance à partir de septembre 2026).

**Site en ligne :** https://diakhitee.github.io/portfolio/ <!-- remplace par l'URL finale une fois le repo renommé si besoin -->

## Contenu du repo

- `index.html` — page d'accueil : présentation et compétences clés
- `ateliers.html` — une vingtaine d'ateliers pratiques réalisés en première année (LAMP, DNS, DHCP, VLAN, pare-feu, Fail2ban, supervision, etc.), avec documentation PDF téléchargeable pour chacun
- `projets.html` — projets plus complets combinant plusieurs compétences à la fois
- `demo-catalogue.html` — aperçu visuel d'un projet web (catalogue d'images en PHP/MySQL)
- `assets/` — feuille de style et scripts du site
- `documents/ateliers/` — documentation PDF de chaque atelier

## Stack technique du site

Site statique en HTML/CSS/JS vanilla, hébergé sur GitHub Pages.

## Compétences illustrées

Administration Windows Server / Linux, Active Directory, GPO, réseaux (VLAN, DNS, DHCP, routage), sécurité (pare-feu, Fail2ban, audit de conformité, ISO 27001, OWASP), virtualisation (VMware, Docker), scripting (Python, Bash, PowerShell), et développement web (PHP, MySQL).
