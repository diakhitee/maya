Catalogue d'images — gestion sans stockage en base

Petite application web développée en environnement local (UwAmp), sous la forme d'un catalogue d'anime. L'objectif technique : afficher des images dans une galerie sans les stocker directement dans la base de données — seuls le nom et le chemin du fichier y sont enregistrés, les fichiers eux-mêmes restant sur le serveur.

PHP MySQL / PDO Apache (UwAmp) Sessions & authentification

Fonctionnalités mises en place

  • Upload d'images avec vérification du type de fichier (jpeg, png, gif)
  • Enregistrement du chemin en base, pas de stockage binaire
  • Authentification (inscription, connexion, déconnexion) avec espace admin protégé par session
  • CRUD complet sur les fiches (création, lecture, modification, suppression)
  • Barre de recherche par titre
  • Pagination de la galerie (6 images par page)
